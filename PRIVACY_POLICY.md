# Privacy Policy — LeetSync

**Last updated:** July 2, 2026

## Data We Collect
LeetSync does **not** collect, store, or transmit any personal data to external servers.

All data is stored **locally** on your device using Chrome's built-in storage API:
- GitHub personal access token (entered by you)
- Repository name
- Solved problem metadata (number, title, difficulty, date)
- Theme and settings preferences
- Achievement progress
- Friend/rival usernames (for the Battle feature)

## Data We Do NOT Collect
- No personally identifiable information
- No browsing history
- No analytics or tracking
- No cookies
- No data sold to third parties

## Third-Party Services
- **GitHub API**: Used solely to push your LeetCode solutions to your own GitHub repository. Your GitHub token is stored locally and never shared.
- **LeetCode**: The extension reads submission data from leetcode.com pages you visit to detect accepted solutions.

## Remote Configuration
The extension periodically fetches a public JSON configuration file from GitHub to check for maintenance status and version updates. No user data is sent during this process.
